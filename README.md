# WebRTCnetmesh

A library for interconnecting several browsers through WebRTC via websockets.

## Usage

A simple usage example:

    import 'package:WebRTCnetmesh/WebRTCnetmesh.dart';

    main() {
      var awesome = new Awesome();
    }
